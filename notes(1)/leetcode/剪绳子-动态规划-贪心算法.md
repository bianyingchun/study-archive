### 贪心算法

尽可能分割成绳子长为 3, 若余下的一段为 1，则让出一个 3，凑成 4

```js
function cuttingRope(n) {
  if (n < 4) return n - 1;
  let times = Math.floor(n / 3);
  const m = n % 3;
  if (m === 1) {
    times - 1;
    return Math.pow(3, times) * 4;
  }
  return Math.pow(3, times) * 2;
}
```
